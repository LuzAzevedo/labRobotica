# Configuração para Manim Slides - ROS2 Aula 1

# Para executar os slides:
# manim-slides ros2_aula1 ROS2Aula1

# Para apresentar os slides:
# manim-slides present ros2_aula1 ROS2Aula1

# Para exportar como HTML:
# manim-slides convert ros2_aula1 ROS2Aula1

# Para exportar como PDF:
# manim-slides convert ros2_aula1 ROS2Aula1 --format pdf
